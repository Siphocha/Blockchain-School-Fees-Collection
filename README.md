# Blockchain-School-Fees-Collection

A blockchain based immutable ledger for managing student fee invoices, payments, and confirmations.

## Features
- Create invoices, record payments, confirm payments, check invoice status.
- Mining of pending transactions using Proof-of-Work.
- View and verify the entire blockchain.
- Persistent storage across program runs.
- Input validation and SHA-256 hashing.

## How to Compile and Run
1. Ensure you have `gcc` and `make` installed.
2. Navigate to the project root.
3. Run `make` to compile.
4. Run `./fee_blockchain` to start the CLI.
